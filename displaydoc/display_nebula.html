Nebula

Abstract 摘要:

An Android game using brain-wave chip and gyroscope to control
一款使用脑波芯片与陀螺仪控制的安卓端游戏


Detail 细节:

Brain wave controls the gravity of the center planet, and the gyroscope controls the movement
脑波芯片控制中心天体的引力，陀螺仪控制其移动


Parameter 参数:

  • Brain-wave Chip: TGMA
  • Gyroscope
  • Android: API 10 +
  • Game Engine: Angle
